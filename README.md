# ProjectPilot AI 🚀

AI-powered project management and developer productivity platform built with Next.js.

## Features

* AI-powered project planning
* Smart developer dashboard
* GitHub integration
* Career roadmap tracking
* AI mentor assistance
* Progress analytics
* Authentication system
* Modern responsive UI

## Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* Prisma
* Clerk Authentication
* Zustand
* PostgreSQL

## Installation

```bash
git clone https://github.com/Yogender-verma/ProjectPilot-AI.git

cd ProjectPilot-AI

npm install

npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```
